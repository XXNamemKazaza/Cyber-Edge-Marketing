Personalisasi

Mode Gelap & Terang

Aplikasi kami beradaptasi dengan gaya Anda. Gunakan mode gelap untuk kenyamanan mata di malam hari, atau mode terang untuk visibilitas maksimal di bawah terik matahari.

RLS

Keamanan Data Lapis Baja

Kami menggunakan protokol Row Level Security (RLS) berstandar kelas dunia. Data setiap sekolah, guru, dan siswa dienkripsi dan diisolasi secara mutlak. Privasi Anda terjaga tanpa kompromi.

Arsip & Backup

Format Eksklusif .edudb
